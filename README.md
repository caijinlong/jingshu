# 乾隆大藏经（龙藏）注音整理

本项目旨在系统整理乾隆大藏经（龙藏）全文，并逐字标注汉语拼音，方便读诵学习。

## 底本

以清代官刻**乾隆大藏经**（又称龙藏）为底本。乾隆大藏经刻于清雍正十一年（1733）至乾隆三年（1738），共 **724 函**、**1669 部**经典，是现存最完整的汉文大藏经版本之一。

## 注音说明

- 格式：每个汉字后紧跟括号内拼音，如 `般(bō)若(rě)波(bō)罗(luó)蜜(mì)多(duō)`
- 佛经专用读音遵循传统读法（详见 [PRONUNCIATION.md](PRONUNCIATION.md)）
- 标点符号不注音，保留原位
